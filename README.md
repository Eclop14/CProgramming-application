# CProgramming-application

Weekly C programming assignments from my university course. Started from the basics and worked my way up to structs and applied programming over 11 weeks.

---

## Structure

```
CProgramming-application/
├── 9/          # Week 9 (Oct 07 - Oct 13)
├── 10/         # Week 10 (Oct 28 - Nov 03)
├── 11/         # Week 11 (Nov 04 - Nov 10)
├── CP1.c
├── CP2.c
├── CP3.c
├── CP4.c       # String practice 1
├── CP4_2.c     # String 4
├── CP5.c       # String 7
├── CP6.c       # String 6
├── CP7.c       # Struct 2
├── CP7-2.c     # Struct 3 (quiz + assignment)
├── CP7-3.c     # Struct 4 (quiz + assignment)
├── CP9.c
├── CP9-2.c
└── CP9-3.c
```

---

## What I learned

- Weeks 1–3: got comfortable with C syntax, I/O, and basic logic
- Weeks 4–5: string manipulation — harder than expected
- Week 6: structs, did both the quiz and assignment versions
- Weeks 9–11: applied everything together

---

## Run it yourself

```bash
gcc -o output CP1.c
./output
```

Tested with GCC on Linux. Should work fine on Mac too.
